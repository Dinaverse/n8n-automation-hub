# ⚡ Sovereign Automation Hub: n8n Workflows

This repository hosts a collection of autonomous **n8n** workflow definitions used to orchestrate and automate a distributed security and AI laboratory.

## 🚀 Key Workflows

### 🤖 Autonomous Executor
- **`n8n_autonomous_executor.json`**: A high-level workflow designed to execute tasks autonomously based on incoming triggers and AI analysis.

### 🛠️ Pro Engineering Agent
- **`n8n_pro_engineering_agent.json`**: Specialized engineering logic for managing complex infrastructure tasks and automated troubleshooting.

### 🔗 Master Integration
- **`n8n_master_integrated.json`**: The centralized hub connecting various services (AI nodes, security tools, and notification systems).

### 📋 Task Coordination
- **`n8n_task_coordinator_reference.json`**: A reference workflow for managing task queues and priority-based execution across the node network.

## 🏗️ Setup
To use these workflows:
1. Install [n8n](https://n8n.io/) (self-hosted Docker recommended).
2. Import the desired `.json` file into your n8n instance.
3. Configure the necessary credentials (API keys, SSH, etc.) within the n8n UI.
